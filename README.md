# quantum-cryptography-notebooks-tr
Kuantum şifreleme ve bilgi teorisi ders notlarının Türkçeleştirilmiş interaktif Jupyter Notebook (.ipynb) arşivi. Temel kuantum araçlarından dolanıklığa, QKD protokollerinden cihazdan bağımsız güvenliğe kadar uzanan teorik ve pratik anlatımları içerir. Kuantum bilişim alanında Türkçe kaynak oluşturmayı hedefler.
